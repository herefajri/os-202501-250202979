strace ls - Week_2

1. execve 
Notifikasi pertama dari System Call dengan fungsi untuk menjalankan dan memulai program `ls` agar bisa dijalankan oleh sistem.
2. brk
Untuk menyiapkan dan mengatur batas ruang memory yang ingin dipakai program agar berjalan optimal `brk(NULL)                               = 0x5cc91f048000`.
3. mmap
Untuk membuka ruang memori baru untuk file/library yang dibutuhkan program dan menaruh isi file/file ke memory agar bisa dibaca secepatnya.
4. access
Pengecekan tentang akses ke file tertentu, apakah bisa diakses atau tidak `/etc/ld.so.preload` (hasilnya `(No such file or directory)` ).
5. openat
Untuk membuka file `/etc/ld.so.cache` yang berisi informasi tentang library yang akan dipakai.
6. fstat
Memberikan informasi tentang file yang sudah dibuka, seperti ukuran dan jenisnya `fstat(3, {st_mode=S_IFREG|0644, st_size=35463, ...}) = 0` dan membantu program mengetahui isi file sebelum digunakan.
7. close
Menutup file yang sudah selesai digunakan.
8. openat
Membuka file library lain seperti `libselinux.so.1`, yang mana dibutuhkan program `ls` agar bisa berjalan dengan lengkap.
9. read
Membaca isi file yang baru dibuka dan memastikan agar isi file dapat digunakan dengan format yang diharapkan
